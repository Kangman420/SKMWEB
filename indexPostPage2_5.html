a = a * n % p;

변수 a에는 입력받은 n * n % p 를 최초에 한번 할당한 다음 반복문에서 위의 식을 계속 반복하여 a의 값이 나올 때마다 카운트를 하기 위해 구성한 배열의 a번째 인덱스의 값을 1씩 증가시켜 해당 인덱스의 값이 3이 됐을 때, 즉 식을 반복하여 같은 값이 3번째 나왔을 때 반복문을 종료하고 현재 카운트 배열에서 값이 2 이상인 인덱스의 개수를 체크하는 방식으로 해결했다.
